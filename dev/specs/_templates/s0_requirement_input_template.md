### 4.1 Happy Path

> 節點標注規則：🤖 = 系統自動、👤 = 人工操作、🔄 = 半自動（需人工確認）、🌐 = 外部服務

```mermaid
flowchart TD
    subgraph P1["階段一：{階段名}"]

        B --> C["🤖 {系統做 Z}"]
    end

    subgraph P2["階段二：{階段名}"]
        D["👤 {操作者做 A}"] --> E["🤖 {系統做 B}"]
    end

    subgraph P3["階段三：{階段名}"]
        F["🔄 {管理者確認 C}"] --> G["🤖 {系統做 D}"]
    end

    P1 --> P2 --> P3
```
